Opdracht 1

Wat zijn de sibilings van het "ul" element?

= het "h1" en "p" element


Wat zijn de ancestors van het tweede "li" element?

= het "ul" element (directe ouder), het "body" element (grootouder), het "html" element (overgrootouder)


Wat zijn de descendants van het "body" element?

= het "h1", "p", "ul" element en de drie "li" elementen binnen de "ul"


Hoeveel elementen in een geldig HTML document hebben geen parent?

Alleen het "html" element heeft geen ouder in een geldig HTML document


Heeft elk element één of meerdere child elementen?

Niet elk element heeft child elementen. Bijvoorbeeld de "li" elementen hebben in deze structuur geen children. Andere elementen bv "ul" hebben wel child elementen
