Wat verandert er bij het openen van de gesloten accordion?

De klasse verandert van het accordion van zodra je erop klikt. Zowel de nieuwe actieve als de zopas gesloten accordion verandert van klasse.

Waarom is er meer code in de DOM-tree structuur dan je html bron code?

De extra attributen en klassen worden door JQuery UI aan de oorspronkelijke HTML toegevoegd zodra het script de accordion-functie uitvoert waardoor de uiteindelijke structuur in de DOM-tree uitgebreider is dan de originele HTML-code.
