Antwoorden op de vragen over Globale Variabelen

  • Blijft de data behouden als je de pagina herlaadt?
    Nee. Globale variabelen worden enkel in het geheugen van de huidige pagina bijgehouden. Bij het herladen van de pagina worden ze opnieuw geïnitialiseerd en gaat de data verloren.
  • Blijft de data behouden als je het tabblad sluit en de pagina opnieuw opent?
    Nee. Net zoals bij het herladen, wordt de globale variabele opnieuw geïnitialiseerd. Er wordt niets opgeslagen tussen sessies.
  • Gebruiken twee tabbladen die dezelfde pagina tonen, dezelfde data?
    Nee. Elke pagina/tabblad heeft zijn eigen geheugenruimte. De globale variabele bestaat dus per tabblad afzonderlijk en wordt niet gedeeld.

Antwoorden op de vragen over Session Storage

  • Blijft de data behouden als je het tabblad sluit en de pagina opnieuw opent?
    Nee. Session storage wordt verwijderd wanneer je het tabblad sluit. Bij het heropenen van de pagina in een nieuw tabblad begint de teller opnieuw.
  • Blijft de data behouden als je de pagina herlaadt?
    Ja. Session storage blijft behouden bij het herladen van de pagina zolang je het tabblad niet sluit.
  • Wordt de data gedeeld door verschillende tabbladen als je de pagina meermaals opent?
    Nee. Elk tabblad heeft zijn eigen aparte session storage. De teller begint in elk tabblad vanaf nul.
  • Denk je dat een andere pagina (andere file of url) aan de data geraakt als je ze in hetzelfde tabblad opent?
    Nee. Session storage is gebonden aan zowel het tabblad als de specifieke pagina/URL. Een andere pagina in hetzelfde tabblad heeft geen toegang tot de data van een andere pagina.

Antwoorden op de vragen over Local Storage

  • Blijft de data behouden als je het tabblad sluit en de pagina opnieuw opent?
    Ja, local storage blijft behouden na het sluiten van een tabblad.
  • Blijft de data behouden als je de pagina herlaadt?
    Ja, de data blijft behouden bij het herladen van de pagina.
  • Blijft de data behouden als je de browser afsluit, opnieuw start en de pagina weer opent?
    Ja, local storage is persistent en blijft behouden na het afsluiten en herstarten van de browser.
  • Blijft de data behouden als je de pagina in een andere browser opent?
    Nee, local storage is gebonden aan de specifieke browser. Een andere browser heeft geen toegang tot deze data.
  • Wordt de data gedeeld door verschillende tabbladen als je de pagina meermaals opent?
    Ja, local storage wordt gedeeld tussen tabbladen binnen dezelfde browser en hetzelfde domein.
  • Denk je dat een andere pagina (andere file of url) aan de data geraakt?
    Alleen als de andere pagina op hetzelfde domein (en met dezelfde oorsprong) staat, kan ze aan de local storage data. Een andere file op een ander pad of een andere URL zonder dezelfde oorsprong heeft geen toegang.
